# ms-payment-management

## Microservicio de gestión pagos construido con Clean Architecture usando el scaffold oficial de Bancolombia.

CARACTERÍSTICAS PRINCIPALES

- Clean Architecture (Domain Driven)

- Programación reactiva con Spring WebFlux

- Scaffold Clean Architecture Bancolombia

- Spring Boot

- PostgreSQL

- CRUD con múltiples entidades relacionadas

- Separación clara de responsabilidades

- Arquitectura escalable y testeable
